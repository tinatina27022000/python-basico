## Lista de Exercicios

### 1. Analisador de Texto: Contador de Vogais, Consoantes e Espaços
* **O que treina:** Manipulação de strings, laços de repetição (`for`) e estruturas condicionais (`if/else`).

### 2. Calculadora de Média Escolar com Situação do Aluno
* **O que treina:** Entrada de dados (`input`), conversão de tipos (strings para floats/inteiros) e operadores lógicos.

### 3. Filtro de Listas: Separador de Números Pares e Ímpares
* **O que treina:** Criação e manipulação de listas (arrays), método `.append()` e o operador de módulo (`%`).

### 4. Conversor de Temperaturas Personalizado (Celsius, Kelvin e Fahrenheit)
* **O que treina:** Criação de funções personalizadas (`def`), passagem de parâmetros e operações matemáticas básicas.

### 5. Validador de Senhas Simples
* **O que treina:** Verificação de tamanho (função `len()`), iteração sobre caracteres e uso de métodos nativos de string (como `.isdigit()` ou `.isalpha()`) para garantir que a senha tenha letras e números.
